◆ take_char_while0()

template<typename Fn >
requires (Predicate<detail::PureT<Fn>, char>)
constexpr auto comb::take_char_while0 ( Fn &&  fn)
constexpr

constructs a TakeCharWhile<Fn, true>

Note: this function is different from take_while0. The latter consumes another parser passed to it, and produces a std::vector<ValueType>. For characters, this function is more efficient, since no allocation occurs, and a direct substring of the input is returned.

Usage:

auto p = comb::take_char_while0(comb::isdigit);
p.parse("123abc"); // ("123", "abc")
p.parse("abc"); // ("", "abc")

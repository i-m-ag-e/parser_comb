constexpr

Combinator that returns a default fallback on failure.

Internally implemented as comb::alt(p, comb::pure(def)), where p is the parser and def is the fallback value. Therefore, d must have the same type as the type the parser produces.

Usage:

auto p = comb::optional_or(comb::map_to_value(comb::stringP("true"), true), false);
auto r1 = p.parse("true"); // (true, "")
auto r2 = p.parse("123;"); // (false, "123;")

Detailed Description

template<Predicate< char > Fn>
struct comb::Satisfy< Fn >

Parser that matches a character satisfying a given predicate.

The predicate must be a callable that takes a char and returns a bool.

Using <cctype> predicates can be a bit of a hassle due to their overloads. To use them with satisfy, there are <cctype> wrappers given in parser_comb/CType.h. See <CType.h> for more details.

Usage:

auto p = comb::satisfy(comb::isdigit);
p.parse("123"); // ('1', "23")
p.parse("abc"); // std::nullopt

Detailed Description

template<ParseRule P, Predicate< typename P::ValueType > Fn, bool AllowZero>
struct comb::TakeWhile< P, Fn, AllowZero >

Combinator that executes a parser as many times as it succeeds and the

consumed value satisfies a predicate.

Allocates on the heap and not constexpr compatible before C++20.

Usage given with construction functions comb::take_while0 and comb::take_while1.

Template Parameters
Pparser to execute
Fnpredicate that checks the parsed value
AllowZeroflag to allow or disallow zero executions of a parser

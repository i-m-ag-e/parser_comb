Detailed Description

template<ParseRule P, typename T>
struct comb::MapToValue< P, T >

Parser that always returns a constant on success.

The parser executes the internal parser P, if it succeeds, returns the constant passed during construction.

Usage:

auto p = comb::map_to_value(comb::stringP("true"), true);
auto r1 = p.parse("true;"); // (true, ";");
auto r2 = p.parse("false"); // std::nullopt

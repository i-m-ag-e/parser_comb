Detailed Description

A parser that matches a specific character.

This parser fails if the input is empty or if the first character does not match the specified character.

Usage:

auto p = comb::charP('a');
p.parse("abc"); // ('a', "bc")
p.parse("xbc"); // std::nullopt
p.parse(""); // std::nullopt

Detailed Description

template<ParseRule P, Predicate< typename P::ValueType > Fn, bool AllowZero>
struct comb::SkipWhile< P, Fn, AllowZero >

Combinator that executes a parser as long as it succeeds and the predicate returns true, discarding all results.

Usage given with construction functions comb::skip_while0 and comb::skip_while1.

Template Parameters
PParser to execute
Fnpredicate that checks the parsed value

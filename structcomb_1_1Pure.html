Detailed Description

template<typename T>
struct comb::Pure< T >

Parser that always succeeds and returns the provided value without consuming any input.

Usage:

comb::pure(42).parse("abc"); // (42, "abc")
comb::pure(std::string("hello")).parse(""); // ("hello", "")
